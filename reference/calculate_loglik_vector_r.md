# Calculate the Log-Likelihood Vector (R Helper)

Calculate the Log-Likelihood Vector (R Helper)

## Usage

``` r
calculate_loglik_vector_r(
  y,
  current_pars,
  spec,
  model_type = "univariate",
  diagnostics = NULL,
  iteration = NULL,
  state = NULL,
  verbose = FALSE
)
```

## Arguments

- y:

  y

- current_pars:

  current_pars

- spec:

  spec

- model_type:

  model_type

- diagnostics:

  diagnostics

- iteration:

  iteration

- state:

  state

- verbose:

  verbose
