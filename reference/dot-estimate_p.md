# Estimate geometric distribution parameter p for stationary bootstrap

Estimate geometric distribution parameter p for stationary bootstrap

## Usage

``` r
.estimate_p(x, p_method, block_length, model_func, score_func)
```

## Arguments

- x:

  x

- p_method:

  p estimation method

- block_length:

  Expected block length

- model_func:

  Model function for k fold cross validation

- score_func:

  Score function for k fold cross validation

## Value

p
