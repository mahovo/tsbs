# Record Statistics for a Bootstrap Replicate

Record Statistics for a Bootstrap Replicate

## Usage

``` r
record_replicate_stats(diagnostics, replicate_idx, bootstrap_matrix)
```

## Arguments

- diagnostics:

  A `tsbs_diagnostics` object.

- replicate_idx:

  Integer, which bootstrap replicate.

- bootstrap_matrix:

  The bootstrap series matrix.

## Value

Updated diagnostics object.
