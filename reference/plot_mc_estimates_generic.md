# Plot Monte Carlo Estimates (Generic)

Create histograms and scatter plots for MC estimates. Works for both
CGARCH and DCC results.

## Usage

``` r
plot_mc_estimates_generic(mc_result, model_type = "CGARCH")
```

## Arguments

- mc_result:

  Result from Monte Carlo study

- model_type:

  Model type identifier for titles

## Value

List with plotly objects
