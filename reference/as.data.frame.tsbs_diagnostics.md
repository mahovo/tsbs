# Convert Diagnostics to Data Frame

Convert Diagnostics to Data Frame

## Usage

``` r
# S3 method for class 'tsbs_diagnostics'
as.data.frame(x, ..., what = c("stats", "blocks", "all"))
```

## Arguments

- what:

  Character specifying what to convert: "blocks", "stats", or "all".

- diagnostics:

  A `tsbs_diagnostics` object.

## Value

A data frame.
